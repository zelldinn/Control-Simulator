# Control-Simulator
WORK IN PROGRESS - This project aims to simulate a rocket launch that can follow the supplied criteria to obtain a reproducible launch and landing, both of which are autonomously adaptable to changing external launch and flight conditions.

## Criteria (for creator's personal reference, these are not organised):
maintains relatively low tilt (stable vertical flight and descent - perhaps using a gyroscope) 
can land in a state that makes it reusable for launching (using a falcon-9 style descent with a rotatable exhaust nozzle to direct it, and the ability to autonomously turn it on and off based on altitude and angle measurements to descend carefully) 
lands completely stable once contacting the floor, without falling sideways or bumping on the floor strongly - perhaps by releasing landing legs or some catching mechanism
